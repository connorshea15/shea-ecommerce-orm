# shea-employee-database

## Description 

This project provides a robust backend for an ecommerce site. It contains relational databases featuring categories, products within those categories and tags to better define those products for user searches. 


## Built With
* Javascript
* MySQL
* Node.js
* ES6
* Sequelize

## Installation

In order to run this application, you must run the following commands:

* npm install inquirer

## Usage 

* Navigate to the root directory of the project in your MySQL shell and type 'source db/schema.sql' in order to create the ecommerce database

* Navigate the root directory of your project in your integrated terminal and type 'npm run seed' in order to create tables and populate those tables with your test data

* Initiate the application by running the command:

node start

* Use Insomnia core to test your routes inputting and outputting json formatted data
